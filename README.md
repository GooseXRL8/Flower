# FlowerLove 🌸

## Um Espaço Romântico e Privativo para Casais

FlowerLove é uma plataforma dedicada a casais que desejam celebrar e eternalizar sua história de amor. Crie um espaço único para guardar memórias, colecionar fotos em formato polaroid, acompanhar o tempo juntos e compartilhar sentimentos em cartões personalizados.

### Funcionalidades
- **Contador Preciso**: anos, meses, dias, minutos e segundos juntos.
- **Baú de Lembranças**: registre viagens, jantares, passeios e piadas internas.
- **Mural de Polaroids**: galeria vintage com registros memoráveis.
- **Gerador de Cartões**: compartilhe digitalmente ou imprima.

### Como rodar
\`\`\`bash
pnpm install
pnpm dev
