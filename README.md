# FlowerLove 🌸
### Um Espaço Romântico e Privativo para Casais

O **FlowerLove** é uma plataforma dedicada a casais que desejam celebrar e eternizar sua história de amor. Mais do que um site, é um espaço seguro para guardar memórias especiais, colecionar fotos em formato polaroid, acompanhar o tempo juntos e compartilhar sentimentos.

![Página Inicial do FlowerLove](docs/screenshots/landing.webp)

---

## ✨ Experiência Sensorial
O FlowerLove valoriza os pequenos detalhes diários:

- **Contador Preciso**: anos, meses, dias, minutos e segundos juntos.
- **Baú de Lembranças**: registre viagens, jantares, passeios e piadas internas.
- **Mural de Polaroids**: galeria organizada em formato polaroid.
- **Cartões de Celebração**: gere cartões para compartilhar digitalmente ou imprimir.

---

## 🛠️ Funcionalidades Principais

| Recurso | Descrição |
| :--- | :--- |
| **Contador** | Marcação precisa do tempo juntos. |
| **Baú de Memórias** | Registro de datas, locais, fotos e sentimentos. |
| **Mural Polaroid** | Galeria visual com upload simples. |
| **Cartões** | Cartões personalizados com o contador da relação. |

---

## 🚀 Como Começar
1. Instale dependências com `pnpm install`.
2. Configure o Supabase se necessário.
3. Rode com `pnpm dev` e abra no navegador.
